# frontend

Vue 3 + Vite + Element Plus 管理后台。

## 已实现

- 管理员登录
- 项目列表
- 创建项目
- 单项目 / 全项目手动同步
- 项目详情
- 最近交易信号列表
- 最近指标列表

## 开发启动

```bash
cd /home/zzzgggwww/openclaw-work/longroot-app/frontend
npm install
npm run dev
```

默认地址：`http://127.0.0.1:5173`

可选环境变量：

- `VITE_API_BASE`：覆盖后端 API 地址，默认 `http://127.0.0.1:3000/api`

> 启动前请先把后端服务跑起来。
